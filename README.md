# Phishing Awareness Web App

This is a web-based cybersecurity awareness quiz designed to educate users about phishing attacks and online safety.

## Features
- User awareness quiz
- Pre-test and post-test
- Admin dashboard
- User consent and results pages

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite database

## How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open the browser at:
   http://127.0.0.1:5000
